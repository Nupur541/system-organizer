# Downloads Folder Organizer

## Problem Statement
My Downloads folder often becomes cluttered with files of different types such as images, documents, videos, and music. This makes it difficult to find important files. This command-line utility helps organize files automatically into categorized folders.

## How to Run
1. Ensure Python 3 is installed on your system.
2. Open terminal inside the project folder.
3. Run the following command:

```bash
python main.py "C:\Users\Nupur\Downloads"
